# Margaret

Margaret is a practical, backend-neutral, biased offline renderer core in Rust.

## Repository setup

This repository uses the files in `primer/` as the coding contract for authoring and editing code.

Before changing code, read:
- `primer/rust-primer.md`
- `primer/rust-primer-examples.md`
- `primer/vulkan-primer.md`
- `primer/vulkan-primer-examples.md` (when Vulkan work begins)

## Tooling

- Rust stable
- `cargo fmt`
- `cargo clippy --workspace --all-targets`
- `cargo test --workspace`
